#initialize(name: nil, gem_version: nil, files: [], generate_output: true, generate_output_flag: false, output_dir: nil, fail_on_warning: false, fail_on_undocumented_objects: false, show_public: true, show_protected: false, show_private: false, hide_private_tag: false, readme: nil, markup: nil, template: nil, template_path: nil, format: nil, options: [], stats_options: []) ⇒ Yardoc

Create the template settings for the Yardoc template.

Parameters:

  • name (String) —

    Name of the tool to create. Defaults to DEFAULT_TOOL_NAME.

  • gem_version (String, Array<String>) —

    Version requirements for the yard gem. Defaults to DEFAULT_GEM_VERSION_REQUIREMENTS.

  • files (Array<String>) —

    An array of globs indicating the files to document.

  • generate_output (Boolean) —

    Whether to generate output. Setting to false causes yardoc to emit warnings/errors but not generate html. Defaults to true.

  • generate_output_flag (Boolean) —

    Whether to create a flag --[no-]output that can control whether output is generated. Defaults to false.

  • output_dir (String, nil) —

    Output directory. Defaults to "doc".

  • fail_on_warning (Boolean) —

    Whether the tool should return a nonzero error code if any warnings happen. Defaults to false.

  • fail_on_undocumented_objects (Boolean) —

    Whether the tool should return a nonzero error code if any objects remain undocumented. Defaults to false.

  • show_public (Boolean) —

    Show public methods. Defaults to true.

  • show_protected (Boolean) —

    Show protected methods. Defaults to false.

  • show_private (Boolean) —

    Show private methods. Defaults to false.

  • hide_private_tag (Boolean) —

    Hide methods with the @private tag. Defaults to false.

  • readme (String, nil) —

    Name of the readme file used as the title page, or nil to use the default.

  • markup (String, nil) —

    Markup style used in documentation. Defaults to "rdoc".

  • template (String, nil) —

    Template to use. Defaults to "default".

  • template_path (String, nil) —

    The optional template path to look for templates in.

  • format (String, nil) —

    The output format for the template. Defaults to "html".

  • options (Array<String>) —

    Additional options passed to YARD

  • stats_options (Array<String>) —

    Additional options passed to YARD stats
